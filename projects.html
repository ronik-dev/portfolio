Backup for Network Appliance Configurations

[GOAL]: Use Ansible to automate the backup of various network appliances configurations.
[CONTEXT]: The device to backup ranged from various Cisco switches to Dell switches mounting DELL OS 10. The ansible machine needed to be able to authenticate on these machine and execute a backup of the configuration. The credentials had to be safely stored. The interaction with the network appliances was made via API when possible, via ssh otherwise.
